Proyecto de Registro y Gestión de Productos
Este proyecto es una aplicación que permite a los usuarios registrarse e iniciar sesión para acceder a una tabla de productos. Los usuarios pueden agregar, eliminar y calcular precios con descuento para los productos en la tabla.

Cómo funciona el proyecto

Inicio de sesión / Registro
Al iniciar la aplicación, el usuario debe registrarse primero si es un nuevo usuario.
Si el usuario ya está registrado, puede iniciar sesión con su nombre de usuario y contraseña.
Si se ingresan credenciales incorrectas o los campos están vacíos, se mostrará un error indicando que el usuario no está registrado o que los campos están incompletos.

Tabla de Productos
Después de iniciar sesión o registrarse correctamente, se abrirá la tabla de productos.
En la tabla, los usuarios pueden realizar las siguientes acciones:
Agregar productos: Los usuarios pueden ingresar información sobre nuevos productos, como nombre, precio, etc., y agregarlos a la tabla.
Eliminar productos: Los usuarios pueden seleccionar uno o más productos de la tabla y eliminarlos.
Calcular precios con descuento: Los usuarios pueden aplicar descuentos a los precios de los productos y ver los precios actualizados en la tabla.

Maqueta(mockup)
Se ha proporcionado una maqueta visual de la aplicación que muestra cómo se vería la interfaz de usuario y las funcionalidades principales. Puedes encontrar la maqueta en el archivo mock.drawio de este repositorio.

Instrucciones de instalación
Clona o descarga este repositorio en tu máquina local.
Abre el archivo main.py en tu editor de código o IDE preferido.
Ejecuta el archivo main.py para iniciar la aplicación.
¡Ahora puedes comenzar a explorar y utilizar la aplicación de registro y gestión de productos!
